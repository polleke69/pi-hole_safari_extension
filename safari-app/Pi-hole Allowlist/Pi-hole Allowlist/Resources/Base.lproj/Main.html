Extension not listed in Safari?

Local / unsigned builds are ignored by Safari until you allow them (once per Safari session):

  1. Install or run Pi-hole Allowlist.app (from Xcode: Product > Run) at least once.
  2. In Safari, open Settings > Advanced and turn on “Show features for web developers” (or the older “Show Develop menu in menu bar”).
  3. Then Settings > Developer and check “Allow unsigned extensions”. This resets when you quit Safari.
  4. Open Settings > Extensions, find Pi-hole Allowlist in the list, and enable the checkbox.
